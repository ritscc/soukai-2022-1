サーバ管理統括として以下の三つを統括とする．
\begin{itemize}
    \item CentOS 7の跡継ぎの調査

9月13日時点ではUbuntuに移行する予定である

\subsection*{各種ミドルウェアの更新}

\subsection*{管理，運用の引き継ぎを行う}
